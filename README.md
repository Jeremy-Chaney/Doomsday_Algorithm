# Doomsday_Algorithm
Program that will help practice the doomsday algorithm to calculate the day of the week for any future or past dates

for a full explanation of the doomsday algorithm, watch the Numberphile video:
https://www.youtube.com/watch?v=z2x3SSBVGJU

For help with the different setting in the script, run in command line:
python3.7 dday_calc.py help
